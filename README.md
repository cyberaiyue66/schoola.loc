# School Website Project (PHP Practice)

Учебный проект школьного сайта, созданный в рамках изучения основ языка PHP. Проект демонстрирует базовые принципы построения динамических веб-страниц и разделения логики и представления.

## 🚀 Функционал проекта
В проекте реализованы следующие модули:
* **Динамическое приветствие:** Сайт меняет текст приветствия в зависимости от времени суток (Утро/День/Вечер/Ночь).
* **Генератор таблицы умножения:** Скрипт `table.php` генерирует таблицу с настраиваемым количеством строк, столбцов и цветовой схемой через GET-запрос.
* **Школьный калькулятор:** Простая форма для арифметических вычислений.
* **Обратная связь:** Форма контактов с динамическим вычислением лимита объема данных (`post_max_size`) в байтах.
* **Универсальная навигация:** Меню сайта генерируется через PHP-функцию `drawMenu`, поддерживающую вертикальное и горизонтальное отображение.

## 🛠 Технологии
* **PHP 8.x** (генерация контента, работа с массивами, функции)
* **HTML5 / CSS3** (блочная верстка, фиксированное меню)

## 📂 Структура файлов
* `index.php` — Главная страница с динамическим приветствием.
* `about.php` — Информация о проекте.
* `contact.php` — Страница контактов и формы обратной связи.
* `table.php` — Скрипт отрисовки таблицы умножения.
* `calc.php` — Калькулятор.
* `style.css` — Основные стили оформления.
